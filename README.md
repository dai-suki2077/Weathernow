**Requirements**
- Android Studio latest version
- JDK 8
- Target Android SDK 35
- Supports API Level +24
- Material Components 1.10.0

**Highlights**
- Use [Open-Meteo] API
- Use Material Design 2
- Responsive UI
- Background Lofi Music
- Uses locale Database
